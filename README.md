# Business Sales Performance Analytics

## Objective
Analyze sales data to identify category performance, regional performance, sales patterns, and profitability.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
1. Technology category generated high sales.
2. West region performed strongly.
3. Sales varied across categories and states.
4. Top sub-categories contributed significantly to revenue.
5. Profitability differed among categories.

## Recommendations
1. Increase marketing for high-performing categories.
2. Improve performance in low-sales regions.
3. Maintain inventory for top-selling products.
4. Optimize discount strategies.
5. Expand successful regional strategies.
